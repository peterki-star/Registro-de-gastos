# Registro de Gastos

Sistema desenvolvido em Python para registrar, organizar e acompanhar gastos pessoais de forma prática e intuitiva.

## Funcionalidades

- Cadastro de gastos
- Edição e exclusão de registros
- Categorização de despesas
- Comparação de gastos com salário
- Visualização mais organizada das finanças
- Controle de gastos fixos e variáveis

## Objetivo do projeto

Este projeto foi criado com o objetivo de praticar programação em Python e desenvolver uma aplicação útil para o controle financeiro pessoal.

## Tecnologias utilizadas

- Python
- Streamlit
- Pandas
